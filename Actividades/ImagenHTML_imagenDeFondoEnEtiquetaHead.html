Imagen de fondo

Puedes especificar imágenes de fondo
para cualquier elemento HTML visible.
En este ejemplo, la imagen de fondo
se especifica para el elemento p.
De forma predeterminada, la imagen de fondo
se repetirá en la(s) dirección(es)
donde es mas pequeño que el elemento
donde se especifica. (Intente cambiar el tamaño de
ventana del navegador para ver como se comporta
la imagen de fondo.)
