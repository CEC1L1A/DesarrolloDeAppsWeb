El elemento picture

Cambie el tamaño del navegador para ver diferentes versiones de la imagen cargándose en diferentes tamaños de ventana gráfica. El navegador busca el primer elemento de origen donde la consulta de medios coincide con el ancho de la ventana gráfica actual del usuario. y obtiene la imagen especificada en el atributo srcset.

El elemento img es necesario como ultima etiqueta secundaria del bloque de declaracion de imagen. El elemento img se utiliza para proporcionar compatibilidad con versiones anteriores de los navegadores que no admiten el elemento de imagen o si ninguna de las etiquetas de origen coincide.
